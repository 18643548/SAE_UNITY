# SAE_UNITY
La version d'Unity utilisée pour le projet est la 2022.3.20f1
Les touches d'actions sont :

Gameplay :
Maj : augmenter la vitesse
Ctrl : diminuer la vitesse
z : pivot bas
s : pivot haut
q : vrille gauche
d : pivot droit
a : rotation gauche
e : rotation droit

Musique :
k : pause
j : précédent
l : suivant
